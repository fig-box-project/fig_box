# 无花盒

Figbox 是一个简单快速的后台服务系统，它可以简单地安装在服务器上并简单地运行。
它依赖于Fastapi，所以它非常之简单和易于上手，即使你不懂得编程，你也可以利用已建打造好的模组来为你建立你的服务。

## 如何开始它？

由于使用了Fastapi你可以使用uvicorn来启动它，比如：

```uvicorn app.main:app --port 8080 --host 0.0.0.0 --reload```
